🏥 Hospital Emergency Room Dashboard (Power BI)
📌 Project Overview

The Hospital Emergency Room Dashboard is an interactive healthcare analytics solution developed using Microsoft Power BI to monitor hospital emergency room performance.

The dashboard enables healthcare professionals and hospital management to analyze patient flow, waiting time, admission trends, satisfaction scores, and referral patterns for better operational decision-making.

🎯 Business Problem

Hospitals receive a large number of patients every day, making it difficult to monitor operational efficiency manually.

This dashboard helps management to:

Monitor patient admissions
Track waiting time
Measure patient satisfaction
Analyze referrals
Improve emergency room performance
🎯 Objectives
Analyze emergency room patient visits
Monitor average waiting time
Track patient satisfaction score
Analyze admission status
Identify referral patterns
Improve healthcare decision-making
